## 泛型

可以支持`任何数据类型`

- 不可以是基础数据类型
	- 有哪些
		- boolean
		- char
		- byte short int long
		- float double
	- 对应的包装类可以使用
		- Boolean
		- Char
		- Byte Short Int Long
		- Float Double
